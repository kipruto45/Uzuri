# Uzuri

## Key Features
- Modular university management backend (Django, DRF, Celery)
- Strict RBAC, audit logging, compliance, and MFA enforcement
- Notification system: in-app, email, SMS (Africa's Talking), push (Firebase Cloud Messaging, OneSignal)
- Analytics endpoints and hooks (Google Analytics, Mixpanel) for all major modules (disciplinary, academic leave, notifications, dashboard)
- LMS integration (Moodle API, webhook endpoints, course/event sync)
- Calendar integration (Google Calendar API, webhook endpoints, event sync)
- Webhook support for payments, LMS, calendar, and external services
- Full test coverage (unit/integration)
